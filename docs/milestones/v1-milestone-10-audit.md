Milestone 10 is complete. Connectors are governed, versioned, bounded, SSRF-guarded, event-sourced, and auditable. Sources (S3/ClickHouse/Kafka) ingest rows idempotently into the event pipeline. Reverse-ETL and export materialize and push data to external sinks read-only. Identity resolution is namespaced, deterministic, and reversible. All governance (scopes, budgets, rates, kill switches, audit) reuses the M9 extension host or a native driver reusing the same SSRF client. No new dependencies added.
