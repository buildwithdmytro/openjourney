**Summary**

Milestone 14 is complete. OpenJourney now has a queryable analytics surface built on projection-maintained fact tables, implementing the `§5.11` experimentation & analytics workstream. Reports accept a universal `ReportQuery` shape (time range + granularity + dimensions + filters), backward-compatible with point-in-time queries. Over-time reporting uses PostgreSQL `date_trunc` + `generate_series` to bucket facts deterministically (exact-count) with zero-fill. New reports include funnel/deliverability-over-time, cohort/retention analysis by first-seen, audience growth, and cost reporting (cost facts stamped at delivery from channel adapters). Metric definitions are explicit, versioned, and immutable (append-only rows, trigger + REVOKE). Saved reports store report configurations for reuse, guarded by a new `reports:write` scope. The UI reuses the M12 component library and a new shared `Chart` primitive (line/bar/funnel/sparkline inline SVG, theme-token-styled) promoting the prior `Overview.SimpleSparkline` and `Reports.FunnelBars` components. An Analytics dashboard section fetches over-time/retention/growth/cost charts from live endpoints and supports save/load. The Overview section now displays real funnel-over-time data instead of synthesized data. All 22 requirements (19.0–19.10) are complete with verifying tests. All 628 Go tests, 273 web tests, and 30 SDK tests pass. No new dependencies added. Analytics milestone complete.
