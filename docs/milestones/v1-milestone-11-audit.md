Milestone 11 is complete. In-app messaging rides the shared delivery pipeline through the native `inapp` adapter, persisting to `inapp_messages`. Web push is a VAPID wake signal under the existing `push` channel. The public client edge is token-authenticated and IDOR-safe. Engagement is event-sourced: reports emit `message.*` events processed by the projector which stamps display-state. Display-state is written only by the projector. Frequency, consent, and suppression apply uniformly. All governance (scopes, suppression, fatigue) reuses existing machinery unchanged. No new dependencies added. All 584 Go tests, 33 web tests, and 14 SDK tests pass.
